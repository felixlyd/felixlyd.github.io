GRANT CREATE SESSION, CREATE TABLE,unlimited tablespace TO nfse;

      update不提交也会锁表

      java代码或者数据库客户端连接数据库后，做update，不做commit，会锁表。等待第一个update在commit或者rollback后，后续的update才能操作。
